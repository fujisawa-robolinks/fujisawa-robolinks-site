# アルバム画像の配置ルール

このフォルダ配下のサブディレクトリを、HP上で1つのスライド枠として表示します。

例:

- `public/album/2026-04_定期練習/01.jpg`
- `public/album/2026-04_定期練習/02.jpg`
- `public/album/2026-04_オフショット/01.jpg`
- `public/album/2026-04_大会/01.webp`

表示ルール:

- サブディレクトリ名がカードタイトルになります
- サブディレクトリ内の画像が自動で切り替わります
- 対応形式は jpg / jpeg / png / webp / gif

推奨:

- ファイル名は半角英数字とハイフンのみ
- 画像サイズは横幅1600px以下を目安
- 形式は jpg / png / webp
